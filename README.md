# PHP-Tasker

## Quick start

- Environment
```shell
cp .env .env.local
```

- Composer
```shell
composer install
```

- Migrations (optional)
```shell
php bin/console doctrine:migrations:migrate
```

- Docker (optional)
```shell
docker-compose -f docker/docker-compose.yml up --build -d
```
